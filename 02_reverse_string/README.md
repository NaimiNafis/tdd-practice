# Exercise 02: reverse_string

## The function is already implemented

Open [reverse_string.py](reverse_string.py) and read it.

```python
reverse_string("hello")         # → "olleh"
reverse_string("Hello, World!") # → "!dlroW ,olleH"
reverse_string("a")             # → "a"
reverse_string("")              # → ""
```
